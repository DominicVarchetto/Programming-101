What is HTML?

HTML is the standard markup language for documents that are designed to be displayed in a web browser. It stands for HyperText Markup Language. You will use HTML when creating webpages like "Getting to Know Each Other" and "Hello World". It is very important that you set the document up correctly when in VS Code.

Explanation of each line of code:

  • !DOCTYPE HTML - what you want the document to be.
  • HTML lang=en - specifies the language of the elements content. EN = English.
  • Meta charset = "UTF-8" - used to specify the character encoding used in an HTML document. The HTML5 encourages developers to use the UTF-8 character set which covers almost all of the characters and symbols in the world.
  • Title - what the web address will be titled
  • H1 - heading of the section.
  • p - paragraph under the heading.
